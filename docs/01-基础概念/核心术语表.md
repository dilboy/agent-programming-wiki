# 核心术语表

## A

| 中文 | 英文 | 解释 |
|------|------|------|
| Agent | Agent / 智能体 | 由 LLM 驱动、能自主感知环境并执行行动的系统 |
| ACI | Agent-Computer Interface | Agent 与计算机/人类交互的接口设计 |
| 自主性 | Autonomy | Agent 无需人类持续干预而独立运行的能力 |

## C

| 中文 | 英文 | 解释 |
|------|------|------|
| 上下文窗口 | Context Window | LLM 单次能处理的最大 token 数量 |
| 上下文学习 | In-Context Learning | LLM 从 Prompt 提供的示例中学习的能力 |

## F

| 中文 | 英文 | 解释 |
|------|------|------|
| 函数调用 | Function Calling | LLM 生成结构化输出以调用外部函数/工具的能力 |
| 少样本提示 | Few-Shot Prompting | 在 Prompt 中提供少量示例以引导 LLM 输出 |

## H

| 中文 | 英文 | 解释 |
|------|------|------|
| 人在回路 | Human-in-the-Loop | 在关键决策点引入人类审核和干预的设计模式 |

## L

| 中文 | 英文 | 解释 |
|------|------|------|
| LLM | Large Language Model | 大语言模型，如 GPT-4、Claude、Gemini 等 |
| LangChain | LangChain | 构建 LLM 应用的流行 Python/JS 框架 |
| LangGraph | LangGraph | LangChain 团队推出的状态机编排框架 |

## M

| 中文 | 英文 | 解释 |
|------|------|------|
| MCP | Model Context Protocol | 标准化模型与外部工具/数据源交互的开放协议 |
| 记忆 | Memory | Agent 保留和检索过去交互信息的能力 |
| 多 Agent 系统 | Multi-Agent System | 多个 Agent 协作完成任务的系统架构 |

## P

| 中文 | 英文 | 解释 |
|------|------|------|
| Prompt | 提示词 | 输入给 LLM 的指令文本，决定输出质量 |
| Prompt 工程 | Prompt Engineering | 设计和优化 Prompt 以获得更好输出的技术 |
| 并行化 | Parallelization | 同时执行多个独立任务以提高效率的模式 |

## R

| 中文 | 英文 | 解释 |
|------|------|------|
| ReAct | ReAct | Reasoning + Acting，推理与行动交替的 Agent 模式 |
| RAG | Retrieval-Augmented Generation | 检索增强生成，结合外部知识库回答问题 |
| 路由 | Routing | 根据输入特征将请求分发到不同处理路径的模式 |
| 推理 | Reasoning | LLM 分析问题、推导结论的认知过程 |

## S

| 中文 | 英文 | 解释 |
|------|------|------|
| 系统提示 | System Prompt | 定义 LLM 角色和全局行为的底层指令 |
| 沙箱 | Sandbox | 隔离的执行环境，限制 Agent 的操作范围 |
| 防护栏 | Guardrail | 约束 Agent 行为的安全规则和检查机制 |

## T

| 中文 | 英文 | 解释 |
|------|------|------|
| 工具 | Tool | Agent 可调用的外部功能（API、函数等） |
| Token | Token | LLM 处理文本的最小单位，通常 1 个中文词 ≈ 1-2 tokens |
| 思维链 | Chain-of-Thought | 引导 LLM 逐步展示推理过程的提示技术 |

## W

| 中文 | 英文 | 解释 |
|------|------|------|
| 工作流 | Workflow | 由开发者预定义执行路径的自动化流程 |
| Wiki 链接 | WikiLink | `[[页面标题]]` 格式的内部链接语法 |

## 常用缩写对照

| 缩写 | 全称 | 中文 |
|------|------|------|
| LLM | Large Language Model | 大语言模型 |
| RAG | Retrieval-Augmented Generation | 检索增强生成 |
| MCP | Model Context Protocol | 模型上下文协议 |
| ACI | Agent-Computer Interface | 智能体-计算机接口 |
| API | Application Programming Interface | 应用程序接口 |
| GUI | Graphical User Interface | 图形用户界面 |
| HITL | Human-in-the-Loop | 人在回路 |
