# 推荐阅读

## 基础论文

1. **ReAct: Synergizing Reasoning and Acting in Language Models** (Yao et al., 2023)
   - 奠定了推理+行动 Agent 架构的基础

2. **Chain-of-Thought Prompting Elicits Reasoning in LLMs** (Wei et al., 2022)
   - 思维链提示的起源

3. **Toolformer: Language Models Can Teach Themselves to Use Tools** (Meta, 2023)
   - 工具学习的里程碑

## 实践指南

- [Building effective agents](https://www.anthropic.com/research/building-effective-agents) — Anthropic 官方指南
- [Patterns for Building LLM-based Systems & Products](https://eugeneyan.com/writing/llm-patterns/) — Eugene Yan
- [The Rise and Potential of Large Language Model Based Agents](https://arxiv.org/abs/2309.07864) — 综述论文

## 书籍

- *Designing Machine Learning Systems* — Chip Huyen（第 7 章涉及 Agent 模式）
- *Building LLM Apps* — Valentina Alto
